# refansa.my.id

My personal website and portfolio, built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Dynamic Content**: Uses MDX for blog posts and project pages.
- **Responsive Design**: Fully responsive layout using Tailwind CSS.
- **Dark Mode**: Supports light and dark modes.
- **SEO Optimized**: Built-in SEO optimization.

## Tech Stack

- [Next.js](https://nextjs.org)
- [TypeScript](https://www.typescriptlang.org)
- [Tailwind CSS](https://tailwindcss.com)
- [MDX](https://mdxjs.com)
