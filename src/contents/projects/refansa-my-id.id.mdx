# refansa.my.id

Situs web pribadi dan portofolio saya, dibangun dengan Next.js, TypeScript, dan Tailwind CSS.

## Fitur

- **Konten Dinamis**: Menggunakan MDX untuk posting blog dan halaman proyek.
- **Desain Responsif**: Tata letak yang sepenuhnya responsif menggunakan Tailwind CSS.
- **Mode Gelap**: Mendukung mode terang dan gelap.
- **SEO Dioptimalkan**: Optimasi SEO bawaan.

- [Tailwind CSS](https://tailwindcss.com)
